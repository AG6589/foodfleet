# FoodFleet Landing Page

A responsive and delicious landing page for FoodFleet, a food delivery service.

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Dynamic Cart**: Interactive cart interface (static demo).
- **Categories & Menu**: Visual display of food categories and menu items.

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript (embedded in HTML/CSS)

## Deployment
Hosted on Vercel.
