# Iron Lady Assignment

This repository contains the solution for the Iron Lady Internal Business Automation technical task.

## Structure

### 1. [Task 1: AI Assistant](./task1-ai-assistant)
Contains the AI prompt engineering logic and documentation for the follow-up message generator.
- **File**: `AI_PROMPT.txt` (The prompt logic)
- **Documentation**: `README.md`

### 2. [Task 2: Internal CRM](./task2-internal-crm)
A complete React + Vite web application for managing leads.
- **Tech Stack**: React, Vite, Vanilla CSS (Glassmorphism Design).
- **Features**:
  - **CRUD**: Create, Read, Update, Delete leads.
  - **AI Bonus**: "Generate Follow-up" button that simulates AI message generation based on lead status.
  - **Responsive**: Fully responsive design for mobile and desktop.

## How to Run

1. Navigate to the CRM folder:
   ```bash
   cd task2-internal-crm
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   npm run dev
   ```

4. Open your browser at the Local URL provided (usually `http://localhost:5173`).
